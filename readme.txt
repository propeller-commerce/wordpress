Bring the convenience of ecommerce to complex business, with a digital sales room that gives everyone the most up-to-date catalogs, pricing, quotes, invoices, and delivery information all in one place.

== Installation ==
Starting with Propeller Ecommerce consists of just two steps: installing and setting up the plugin.

### INSTALL FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’;
2. Search for ‘Propeller Ecommerce’;
3. Activate "Propeller Ecommerce" from your Plugins page;
4. Go to ‘after activation’ below.

### INSTALL MANUALLY

1. Upload the ‘propeller-ecommerce’ folder to the /wp-content/plugins/ directory;
2. Activate the "Propeller Ecommerce" plugin through the ‘Plugins’ menu in WordPress;
3. Go to ‘after activation’ below.

### AFTER ACTIVATION

1. Go to "Propeller" page in the side menu in /wp-admin;
2. Add your API Key and configure your settings
3. You’re done!
